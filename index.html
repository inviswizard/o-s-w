How Our Space Works

  • Secure Storage: Your data is securely stored in encrypted GitHub repositories
  • Private Rooms: Each couple gets their own private space with a unique room code
  • Memory Sharing: Upload photos and create lasting memories together
  • Sweet Notes: Leave loving messages for each other
  • Nicknames: Give each other cute nicknames that only you both see
  • Admin Control: Room creators manage who can join your space
  • One Room Policy: Each user can only be in one room at a time for privacy

Your privacy is our priority. All data is encrypted and stored securely.
